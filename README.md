Репозиторий проекта для финала ВсОШ по информационной безопасности.

Репозиторий содержит материалы творческого проекта:

- Исходный код Telegram-бота (папка code);
- Видеодемонстрацию работы бота.

Тема проекта: «Разработка Telegram-бота для автоматизированного анализа файлов и ссылок на наличие вредоносного контента и фишинговых угроз»
